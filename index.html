DEELNEMEN? DEELNEMEN? DEELNEMEN? DEELNEMEN? DEELNEMEN?
  1. Vind iets om je buurt mee op te lichten 💡
    📽 Projectors (Dia / Overhead / Beamer / …)
    📺 Schermen (TV, Computer, tablet/ ...)
    🎄 Lichtjes (Kerstverlichting / kaarsje / …)

  2. Kies een plek, liefst zodat je buren mee kunnen genieten! 🚗
    🚪 Je raam (Gordijn / papier / …)
    🏠 Je voorbuur zijn gevel (Zeker eerst vragen!)
    🌳 Natuur (Bomen / wolken / …)
  3. Deel het
    1. We willen weten wie er meedoen. Vul onderstaand formulier in. Vertel ons wie je bent.






    2. Stuur ons fotos van het werk, je beamer, je straatkant installatie.
    3. Deel je werk op instagram: #BULP2020
    4. Meer mensen bereiken?
      Stuur een paar dagen voor het evenement, een video door van je werk. Dan zetten we een videostream op, zodat iedereen van de kunst kan genieten. Link volgt later nog.
