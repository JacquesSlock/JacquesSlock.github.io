21/11/2020 // 19:00-22:00
DEELNEMEN? DEELNEMEN? DEELNEMEN? DEELNEMEN? DEELNEMEN?
  1. Vindt iets om je buurt mee op te lichten*
    📽 Projectors (Beamer / Overhead / Dia's / …)
    Schermen (TV / Computer / Tablet/ ...)
    Lichtjes (Kerstverlichting / kaarsje / …)

    *op zoek naar een beamer? mail even naar lucas@nerdlab.be en hij kijkt wat hij voor jouw kan doen.
  2. Kies een plek, liefst zodat je buren mee kunnen genieten!
    Je raam (Gordijn / papier / …)
    Je voorbuur zijn gevel (Zeker eerst vragen!)
    Natuur (Bomen / wolken / …)
  3. Deel het
    1. We willen weten wie er meedoen. Vul onderstaand formulier in. Vertel ons wie je bent.













      * Deelnemers uit Gent kunnen een verassing verwachten op de avond zelf.
    2. Deel je werk op instagram #BULP2020 of stuur ons fotos van het werk, je beamer en/of je installatie. Wij doen dit dan ook.
    3. Meer mensen bereiken?
      Stuur een paar dagen voor het evenement een video door van je werk. Dan zetten wij een videostream op, zodat iedereen van je creatie kan genieten. Link volgt later.
